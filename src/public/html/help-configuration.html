App/Frame level shortcuts

Open a file or directory

  • Open Video: Ctrl + O
  • Open Image Directory: Ctrl + I
  • Open TFRecord Directory: Ctrl + Space

Frame control

  • Next Frame:Right Arrow or E
  • Previous Frame:Left Arrow or Q
  • Duplicate last frame regions: D
  • Delete Frame: Shift + Del
  • Active Learning: Ctrl + L

Tags collection

  • Save Tags: Ctrl + S
  • Export Tags: Ctrl + E

Play control

  • Toggle Play/Pause: Space
  • Toggle Tracking: Ctrl + T

App control

  • Open Developer Console: Ctrl + D
  • Refresh App: Ctrl + R
  • Show Help: Ctrl + H

Region level shortcuts

Region creation tools (rectangular)

  • Rectangular box: R key.
    • Two-point mode: Hold down Ctrl while creating a region
    • Square mode: Hold down Shift while creating a region
  • Template based box: T key.
  • Point surrounding box: P key.
  • Polyline surrounding box: Y key.
  • Exclusive selection: L key to block/unblock regions manipulation.

Region selection & copying

  • Multi-select: Hold down Shift while selecting regions
  • Select next: Tab
  • Select All: Ctrl + A
  • Cut: Ctrl + X
  • Copy: Ctrl + C
  • Paste: Ctrl + V

Region Manipulation with Arrow Keys

VOTT allows you to fine tune the bounding boxes using the arrow keys in a few different ways. While a region is selected:

  • Move region: Hold Ctrl + Direction
  • Shrink region:: Hold Ctrl + Alt + Direction
  • Expand region: Hold Ctrl + Shift + Direction

Styling and Visual manipulations

Tuning visual display of regions:

  • Toggle region background: Ctrl + B

Tags level shortcuts

Tagging:

  • Quick access to tags: 1-9 numeric keys
  • Auto applying a tag: Alt + 1-9 numeric key

Changing Tags

  • Change mapped hotkey: hold Shift + 1-9 numeric key and click on the tag that you want to assign to that hotkey.
  • Delete tag from the list: Ctrl + click on the tag you want removed. It will remove the tag from the list and untag any images that use that tag.
